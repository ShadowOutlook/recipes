**Ингредиенты**
```bash
1. Лаваш
2. Куриное филе
3. Огурцы
4. Помидор
5. Морковь
6. Лук
7. Чикен карри соус
```

**Морковь + Лук => Морковка по корейский**
```bash
1. Морковь на терке
2. Лук мелко нарезать
3. Смешать с добавлением 9% уксуса
4. Соль, перец, имбирь молотый, приправа для моркови по корейский, приправа карри
5. Разогреть всю получившуюся смесь на сковороде с маслом
```

**Чикен карри соус**
```bash
1. Сметана 20% x 1
2. Майонез x 1
3. Карнишоны маринованные x 1
4. Укроп x 0.5
5. Чеснок x 0.3
6. Черный перец x 0.1
7. Яйца x 1
8. Лимонный сок x 0.1
9. Сахар x 0.1
10. Зеленый лук x 0.5
11. Горчица x 0.3
12. Соевый соус 0.1
14. Специи: карри, приправа для куринного бульона, орегано, базилик, тимьян
15. Все смешать и нужно добавить кипеченую воду x 0.2 и взбить блендером
```

**Куриное филе**
```bash
1. Разрезать на части размером половины ладони
2. Желательно отбить для мягкости
3. После обвалять в масле со специями: приправа для курицы или любые другие
4. Другие: хмели сунели, черный перец, французкие травы и т. п.
5. Выложить на противень и готовить 25 мин при 200 град
```
